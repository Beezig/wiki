    Commands

    New commands are one of the huge additions to the user-experience. We added over 40 additional commands, which are explained below. (In alphabetical order)

    Tab-Completion available with BeezigForge!

    Most commands feature tab-completion, which lets you easily enter names or other values. Still, this feature will require the BeezigForge addon!

    All commands¶

    This list is valid for Beezig 6.0

    If you are playing on another version, this information may have changed. Run /beezig to get your version.

    All 'game' or 'mode' terminology used requires the shortcode of the game to be used.

    Example: TIMV, BED, DR, SKY, BP, HIDE

    AddNote TIMV¶

    Adds a small text note to your TIMV notes and hints, viewable by /notes.

    • Syntax: /note [note]
    • Alias: /note, /addnote

    AutoGG¶

    If enabled, you will automatically say gg at the end of a game. You can customize the text, delay and which games to use on. Run /autogg to see current status. More info on AutoGG on the features page!

    • Syntax: /autogg
    • Syntax: /autogg [text/delay] [value]
    • Syntax: /autogg [enable/disable] [shortcode]

    AutoVote¶

    If enabled, we will automatically vote for your favorite maps when you join a lobby. Your favorites are listed via the list argument. After we checked the voting options, we'll go through your favorites top to bottom, meaning the top map will be checked first. To reorder you maps, use the place argument. More info on Autovote on the features page!

    If you have BeezigForge, everything will be done through a GUI instead of chat. Run /autovote

    For chat setup, maps are added in the format mode_map, Spaces are replaced with _

    Example: /autovote add TIMV_Hermit_Sands:_Classic

    • Syntax: /autovote
    • Syntax: /autovote list [mode]
    • Syntax: /autovote [add/remove] [map]
    • Syntax: /autovote place [map] [newPosition]
    • Alias: /autovote, /avote

    Beezig¶

    This command will get you information on your current Beezig installation (Version, Client) It will also indicate if a update is available. The commands argument will instead print all loaded commands.

    • Syntax: /beezig
    • Syntax: /beezig commands
    • Syntax: /beezig reconnect

    BestGame¶

    Calculates which gamemode the most played gamemode is. Use this to find out 'what is their main game?'. This is calculated by using the average top 20 points and comparing it to their points. To get the full list of games, add a second argument (anything). When no player is specified, your stats are calculated.

    TIMV & OITC are currently not calculated/considered! Beezig 6.0-pre

    • Syntax: /bestgame (player) (*)
    • Alias: /bestgame, /bg, /maingame

    Blockstats HIDE¶

    Prints the levels of all hiding-blocks by the player, sorted by total experience. When no player is specified, your stats are calculated.

    • Syntax: /blockstats list (player)
    • Alias: /blockstats, /bs

    ChatReport¶

    Beezig will connect to the Hive Report system and will automatically file a chat-report from ingame. When you run the command, a chatlog will be generated and the report form will be filled. More info on ChatReport on the features page!

    This feature only works on the 5zig version. LabyMod handles ChatComponent packets differently.

    Only the official chat-reasons are available and valid.

    • Syntax: /chatreport [player] [reason(s)]
    • Alias: /chatreport, /reportchat

    CheckPing¶

    Find out the ping of a player. When no player is specified, your ping is shown.

    • Syntax: /checkping (player)
    • Alias: /checkping, /cping

    ClosestToWR DR¶

    Print information on how PB's relate to WR's and what the difference is. Either prints best, worst and average, or all played maps with a second argument (anything). When no player is specified, your stats are calculated.

    • Syntax: /cwr (player) (*)
    • Alias: /cwr, /bestmap

    Daily¶

    Print information on daily points. More info on Daily Points on the features page!

    • Syntax: /daily (mode)

    DeathrunRecords DR¶

    Print Kills & Deaths record.

    • Syntax: /drbest (player) (map)
    • Alias: /drbest, /drrec

    Leaderboard¶

    Print a nice leaderboard. This shows main points, rank and network rank. More info on Leaderboards on the features page!

    • Syntax: /lb [startPos] [endPos] (mode)
    • Alias: /lb, /leaderboard, /leaderboards

    Math¶

    Calculate an expression. Addition, Subtraction, Multiplication, Division and more. You can only invoke methods from the Math class.

    • Syntax: /math [expression]

    Medals¶

    Show the amount of medals of a player. When no player is specified, yours are shown.

    • Syntax: /medals (player)

    MessageOverlay¶

    Just running /msg [player] will toggle the private message mode to player, just like /p does for parties. All your normal messages will get sent in private to player. Doing just /msg will revert to the default state.

    • Syntax: /msg
    • Syntax: /msg [player]
    • Alias: /msg, /tell

    PB DR¶

    Shows the personal best/fastest time time of player on map. No player means your stats will be calculated. If no map is specified, the current active map will be used (if available).

    • Syntax: /pb (player) (map)
